## 4. 差异原因解释
对于部分标的存在的起始日期微小差异（4-13天），经分析主要原因如下：

1.  **上市日与交易日的差异**：ETF基金成立（合同生效）后，通常需要经过一段时间的建仓期（通常为1-3个月，但上市交易可能更快），上市公告书中的“上市日期”是开始在交易所挂牌买卖的日期。但在数据源中，有时最早的行情数据可能因建仓期封闭或初始流动性原因略晚于名义上市日。
2.  **节假日因素**：如果上市日恰逢周五或节假日前夕，实际首个完整交易日的数据可能会顺延。例如 **港股科技ETF (513020)** 要求日期是 2022-01-27（春节前夕），实际数据从 2022-02-08 开始，这正是 2022 年春节假期结束后的第一个交易日（2022年春节假期为1月31日至2月6日，但2月7日可能还未恢复或数据源记录差异），时间点完全吻合。
3.  **数据源收录规则**：Tushare 数据通常记录的是正式产生交易量的首个交易日。

**结论**：
所有标的数据均已成功下载，且覆盖了自实际开始交易以来的完整历史。存在的微小日期差异属于正常的市场交易安排（如春节休市），数据完整性符合策略回测要求。
