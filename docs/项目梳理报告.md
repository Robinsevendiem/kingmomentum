# FoolReveal 项目梳理报告

## 1. 项目概况

本项目是一个完整的 ETF 动量轮动策略回测与分析系统。经过多轮迭代，项目已从最初的脚本集合演变为一个结构清晰、功能完善的 Web 应用。

### 核心功能矩阵
- **策略回测**: 支持自定义参数的历史回测。
- **实盘辅助**: 一键获取 T+1 最新持仓信号。
- **动量分析**: 验证动量因子有效性（倒 U 型曲线）。
- **参数优化**: 基于全量数据的网格搜索与稳健性分析。

## 3. 部署指南 (Streamlit Cloud)

本项目已完全适配 Streamlit Community Cloud 部署。

### 步骤 1: 上传 GitHub
1.  在 GitHub 上创建一个新的仓库（建议设为 Private，因为包含 Token）。
2.  将本地所有文件推送到该仓库。

### 步骤 2: 配置 Streamlit Cloud
1.  登录 [share.streamlit.io](https://share.streamlit.io/)。
2.  点击 **"New app"**。
3.  选择对应的 GitHub 仓库。
4.  **Main file path** 填写 `Home.py`。
5.  点击 **"Deploy"**。

### 步骤 3: 配置 Secrets (可选但推荐)
为了安全起见，建议不要将 Tushare Token 硬编码在代码中，而是在 Streamlit Cloud 的后台配置：
1.  在 App 的仪表盘点击 "Settings" -> "Secrets"。
2.  添加如下内容：
    ```toml
    TUSHARE_TOKEN = "你的真实 Tushare Token"
    ```
3.  系统会自动读取该 Token。

## 4. 后期维护

- **数据更新**: 
    - 每日点击网页上的“更新数据”按钮即可同步最新行情。
    - 无需重新部署代码。
- **参数分析更新**:
    - 如果积累了较长时间的新数据（如半年后），建议本地运行 `python scripts/fine_tune_strategy.py`。
    - 将生成的 `optimization_results.csv` 覆盖 `data/` 目录下的旧文件。
    - 提交并推送到 GitHub，线上应用会自动更新分析报告。
